# Contributing

Thank you for considering a contribution.

## Suitable contributions

- Improve field definitions or documentation
- Add reusable HR analytics examples
- Correct a data-quality or consistency issue
- Add Power BI, Tableau, SQL or Python analyses
- Improve accessibility or chart interpretation
- Add tests without introducing real personal data

## Contribution process

1. Fork the repository.
2. Create a feature branch.
3. Make focused changes.
4. Run `python scripts/validate_repository.py`.
5. Do not commit credentials, real employee information or confidential data.
6. Open a pull request explaining the change and its analytical value.

## Data policy

All contributed records must remain synthetic. Contributions containing real
personal, medical, payroll, applicant or confidential organisational data will
not be accepted.
