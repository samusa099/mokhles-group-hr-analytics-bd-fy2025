# Analysis-Ready CSV Tables

This directory contains the 13 analysis-ready tables used throughout the
Mokhles Group HR Analytics portfolio.

- Files `01`–`12` contain the analytical datasets.
- File `13_Data_Dictionary_FY2025.csv` defines every published field.
- All records are synthetic and intended for learning and portfolio use.
- CSV headers match the schemas documented in
  `../../metadata/kaggle-dataset-metadata.json`.
