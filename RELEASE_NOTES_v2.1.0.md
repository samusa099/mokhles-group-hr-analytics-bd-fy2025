# Mokhles Group HR Analytics — v2.1.0

## BI Compatibility Update

This feature release adds a documented, cross-platform implementation layer for
Power BI, Looker Studio, Tableau, Qlik Sense, Excel Power Query and Metabase.

## New assets

- 10 detailed BI Markdown guides
- 15 BI-ready CSV tables
- Unified 516-record employee dimension
- Date, department and location dimensions
- Relationship mapping file
- KPI definition catalogue
- Six-page dashboard blueprint
- Power BI theme JSON
- Field-name mapping from original headers to snake_case headers
