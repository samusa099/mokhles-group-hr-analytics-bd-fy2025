# Mokhles Group HR Analytics — v2.1.0

## Business Intelligence Compatibility Update

This feature release adds a documented, cross-platform implementation layer for Power BI, Looker Studio, Tableau, Qlik Sense, Excel Power Query/Power Pivot and Metabase.

## New assets

- 11 detailed BI Markdown guides
- Generator for 15 BI-ready dimension and fact CSV tables
- Unified 516-record employee dimension
- Date, department and location dimensions
- Relationship mapping file
- KPI definition catalogue
- Six-page dashboard blueprint
- Power BI theme JSON
- Original-to-BI field-name mapping

## Generate the BI layer

```bash
python scripts/build_bi_ready_layer.py
```
