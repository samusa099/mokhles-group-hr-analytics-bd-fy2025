```text

Type: Text
Current Value: C:\Users\Musa PC\Downloads\Mokhles_Group_HR_Analytics_FINAL_Kaggle_Upload_v2_3_0
```

```text

```

Open Power Query, create a Blank Query, open Advanced Editor, paste the
corresponding `.m` script, and rename the query to match the table.
