# Kaggle Dataset Description

Mokhles Group HR Analytics Demo 2025 is a complete, fully synthetic Bangladesh
HR and People Analytics portfolio.

The project includes original operational-style HR data, professional Excel
reports, a normalized BI layer, consolidated employee and department analytics,
data-quality profiling, a Python notebook, and implementation assets for Power
BI, Excel, Looker Studio, Tableau, Qlik Sense and Metabase.

No real employee, applicant, salary, health or confidential organizational
information is included.
