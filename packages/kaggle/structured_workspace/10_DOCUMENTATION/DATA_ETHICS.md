# Data Ethics and Limitations

- No real employee, applicant, salary, medical or confidential company data
  is included.
- Names, IDs, transactions and organisational outcomes are synthetic.
- The dataset is suitable for education and portfolio demonstrations.
- It must not be used to make decisions about real individuals.
- Diversity fields should be interpreted responsibly and should not be used
  to infer protected characteristics about real people.
- Analytical findings represent a fictional scenario rather than actual
  business performance.
